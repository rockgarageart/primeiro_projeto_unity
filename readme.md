## Descrição do Projeto

Este projeto foi desenvolvido na Unity 6 utilizando Meta XR SDK para criação de uma experiência interativa em realidade virtual.

O ambiente representa um treinamento gamificado de combate a incêndio para funcionários dentro de um contexto de Metaverso educacional.

## Estrutura das Cenas

O projeto contém duas cenas principais:

* `primeiroprojeto.unity`

  * versão inicial do ambiente VR básico

* `projetoavancado.unity`

  * versão interativa final contendo sistema de perguntas, abertura de portas, iluminação dinâmica e interação com o ambiente


## Objetivo

O objetivo é criar uma experiência imersiva onde o usuário aprende através de desafios e interação com portas, perguntas e elementos visuais.

## Funcionalidades

* Ambiente 3D interativo
* Sistema de perguntas e respostas
* Portas que abrem ao acertar respostas
* Feedback visual com iluminação em personagens
* Skybox configurado
* Navegação em primeira pessoa
* Compatível com Meta XR SDK

## Tecnologias Utilizadas

* Unity 6 (6000.3.13f1)
* C#
* Meta XR SDK
* TextMeshPro

## Estrutura do Projeto

* Assets
* Packages
* ProjectSettings

## Dificuldades Encontradas

Durante o desenvolvimento foram enfrentados desafios relacionados à configuração do Meta XR SDK, colisões, interação entre objetos e lógica de abertura das portas utilizando scripts C#.

## Resultado

O projeto demonstra conceitos fundamentais de desenvolvimento XR e interação em ambientes imersivos dentro do contexto de Metaverso.
